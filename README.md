## Development ðŸ’»

Here are the steps to run the project locally.

1. Clone the repository

   ```bash
   git clone https://github.com/cycleccc/twitter-clone.git
   ```

2. Install dependencies

   ```bash
   pnpm i
   ```

3. run next web

   ```bash
   cd apps/web
   pnpm run dev
   ```

4. run nest server
   ```bash
   cd apps/server
   pnpm run dev
   ```

5. run RN app
   ```bash
   cd apps/mobile
   pnpm run start
   ```
